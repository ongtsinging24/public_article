Permanent（永久的）由
	字首 per-（貫穿、完全）、
	字根 -man-（停留、居住） 
	與形容詞字尾 -ent 組合而成。


字面意義為「貫穿始終地停留」，衍生為「永久的、持久的」。
我們可以透過字根 -man- 記憶以下一系列同源單字：
	Permanent（永久的）：	[字首 per- 完全 + 字根 man 停留]
	Remain（留下、保持）：[字首 re- 回、再次 + 字根 main (man 變體) 留下]
	Mansion（公館、大廈）：[源自 manere，原指可以久留的「大型永久住所」]
	Impermanent（非永久的）：[字首 im- 相反 + permanent 永久的]



這三個字首因為拼法極為相似（都由 p、r、e、o 組成），是英文學習者最容易搞混的「魔王級」字首。
核心的區別在於它們的「方向性」與「時間軸」：

* pre-：強調「時間在前」（Before）
* pro-：強調「空間向前」或「支持」（Forward / For）
* per-：強調「穿透空間」或「徹底、到頭」（Through / Completely）

------------------------------
## ⏳ 1. pre- = 在……之前、預先 (Before / In advance)
這個字首完全聚焦於時間的先後。只要看到它，通常代表事情在某個時間點「提早」發生。

* predict (v.) 預測
* 拆解：pre- (預先) + dict (說) → 事情發生前預先說出來。
* prepare (v.) 準備
* 拆解：pre- (在之前) + pare (生產/安排) → 在活動開始之前先安排好。
* prevent (v.) 預防、阻止
* 拆解：pre- (在之前) + vent (走) → 在問題發生之前先走過去擋住。
* prewar (adj.) 戰前的
* 拆解：pre- (在之前) + war (戰爭) → 戰爭爆發之前的。

------------------------------
## 🚀 2. pro- = 向前、往前方；支持 (Forward / Forth / For)
這個字首主要強調空間上的推進、目光向未來延伸，或是立場上的贊同與支持。

* progress (n./v.) 進步、前進
* 拆解：pro- (向前) + gress (走) → 腳步不斷向前走。
* project (n./v.) 專案；投射
* 拆解：pro- (向前) + ject (扔/擲) → 把想法或光線往前方扔出去。
* promote (v.) 晉升、促銷
* 拆解：pro- (向前) + mote (移動) → 讓職位或銷量往前移動。
* pro-environmental (adj.) 支持環保的
* 拆解：pro- (支持) + environmental (環境的) → 立場上支持環保的。

------------------------------
## 🎯 3. per- = 穿過、貫穿、徹底 (Through / Completely)
這個字首強調從頭到尾的穿透感，或是把一件事情做得很徹底、完美。

* permanent (adj.) 永久的
* 拆解：per- (徹底/貫穿) + man (停留) → 狀態貫穿始終都停留在原處。
* perfect (adj.) 完美的
* 拆解：per- (徹底) + fect (做) → 每一處都徹底做好、毫無瑕疵。
* perfume (n.) 香水
* 拆解：per- (穿透) + fume (煙霧) → 讓香氣像煙霧一樣穿透、瀰漫整個空間。
* persecute (v.) 迫害
* 拆解：per- (到底) + secu (跟隨) → 緊跟在後追到底、死纏不放地欺負。

------------------------------
## 👀 終極對比：用同一個字根看懂三者差異
用字根 spect (看) 來對比，就能瞬間秒懂：

* prespect (❌英文無此字，通常用 prospect)
* prospect (n.) 前景、展望
* 解釋：pro- (向前) + spect (看) → 眼睛向前看未來的發展。
* perspect (延伸為 perspective n. 觀點、透視圖)
* 解釋：per- (穿透) + spect (看) → 目光穿透表面、看清事物的本質。

------------------------------

這一組字非常經典！它們的共同核心字根是 -duce（引導、帶領），源自拉丁文動詞 ducere。
只要掌握了「引導」這個核心概念，再搭配不同的字首方向，就能瞬間秒懂這三個字的字源邏輯：
------------------------------
## 🚀 1. produce (v.) 生產、製造

* 字首 pro-：向前、往前方 (Forward)。
* 字根 -duce：引導、帶領 (Lead)。
* 字源密碼：「向前引導出來」。
* 邏輯聯想：把原料或想法往前引導、帶到市場或現實中，讓它具現化，這就是生產、製造。
* 同家族延伸：
* product (n.) 產品（被引導出來的成果）。
   * producer (n.) 製作人、生產者（負責引導生產的人）。

------------------------------
## 📉 2. reduce (v.) 減少、降低、簡化

* 字首 re-：往後、退回 (Back / Down)。
* 字根 -duce：引導、帶領 (Lead)。
* 字源密碼：「往回引導、帶下來」。
* 邏輯聯想：把原本很高的數量或規模，往後、往下引導拉回來，就是減少或降低。
* 同家族延伸：
* reduction (n.) 減少、折扣（引導往下的狀態）。

------------------------------
## 🤝 3. introduce (v.) 介紹、引進

* 字首 intro-：向內、進入內部 (Into / Inside)。
* 字根 -duce：引導、帶領 (Lead)。
* 字源密碼：「引導進入內部」。
* 邏輯聯想：把一個新朋友或新觀念，帶進妳的朋友圈或原有的系統內部，這就是介紹或引進。
* 同家族延伸：
* introduction (n.) 介紹、引言（帶領入門的文字）。

------------------------------
## 💡 延伸擴充：-duce（引導）家族的其他大咖
既然學會了這個字根，以下這兩個高頻字也能順便一網打盡：

* conduct (v./n.) 引導、指揮、行為
* 拆解：con- (共同) + duct (引導) → 把大家聚在一起引導 → 指揮（樂團）或傳導（熱/電）。
* educate (v.) 教育
* 拆解：e- (向外) + duc (引導) + -ate (動詞字尾) → 把孩子內在的天賦和潛能引導出來 → 教育的本質。



deduce 由字首 de- (向下、向外) 和字根 -duce (引導) 組成，表示從已知的事實中「推斷」出結論。

如果想繼續挑戰，我們可以看看另外一組同樣很常混淆的 -cept（拿、取） 家族，例如：accept（接受）、except（除了）、intercept（攔截）。妳想試試看嗎？

這一組字根在多益和各大英文檢定中出題率極高！核心字根 -cept（拿、取、捕獲） 源自拉丁文 capere。
只要想像你的手正在「做拿取的動作」，再搭配字首的方向，就能完美破解這些單字的底層邏輯：
------------------------------
## 📥 1. accept (v.) 接受、認可

* 字首 ac-：源自 ad-，表示「朝向、去」（to / toward）。
* 字根 -cept：拿、取（take）。
* 字源密碼：「朝著自己拿過來」。
* 邏輯聯想：別人遞東西過來，你伸手朝自己拿過來，這就是接受或收下。
* 同家族延伸：
* acceptable (adj.) 可接受的。
   * acceptance (n.) 接受、容忍。

------------------------------
## 🚫 2. except (prep./v.) 除了……之外、排除

* 字首 ex-：向外、出去（out）。
* 字根 -cept：拿、取（take）。
* 字源密碼：「拿出去、拿走」。
* 邏輯聯想：在一堆東西裡面，把其中一個單獨拿出來放到外面，剩下的就是「除了它之外」的。
* 同家族延伸：
* exception (n.) 例外（被拿出去的那個特殊個案）。
   * exceptional (adj.) 卓越的、例外的（優秀到被單獨拿出來誇獎）。

------------------------------
## 🛑 3. intercept (v.) 攔截、截擊

* 字首 inter-：在……之間、中間（between / among）。
* 字根 -cept：拿、取（take）。
* 字源密碼：「在半路中間拿走」。
* 邏輯聯想：球或信件在從 A 傳到 B 的途中，你突然在半路伸出手把它拿走，這就是攔截。
* 同家族延伸：
* interception (n.) 攔截（美式足球中常聽到的「抄截」）。

------------------------------
## 💡 驚喜加碼：還有這兩個高頻字也是同家族！
懂了 -cept 是「拿、取」，下面這兩個生活神級單字你也能秒懂：

* concept (n.) 觀念、概念
* 拆解：con- (共同/完全) + cept (拿/抓) → 大腦把所有的想法完全抓取、匯聚在一起，形成的整體架構 → 概念。
* perceive (v.) 察覺、感知（-ceive 是 -cept 的動詞變體）
* 拆解：per- (徹底/穿透) + ceive (拿/抓) → 感官徹底抓取到外界的訊息 → 察覺、感覺到。

------------------------------

###
dict(說、言語、命令) 

這一組字根超級實用！核心字根 -dict（說、言語、命令） 源自拉丁文動詞 dicere（說）。
只要想像這是一個「開口說話、發表言論」的動作，搭配不同的字首，就能輕鬆解鎖一整串核心單字：
------------------------------
## ⏳ 1. predict (v.) 預測、預言

* 字首 pre-：在……之前 (Before)。
* 字根 -dict：說 (Say)。
* 字源密碼：「在事情發生之前先說」。
* 邏輯聯想：在未來的事情還沒發生之前，你就提早開口說出來，這就是預測。
* 同家族延伸：
* predictable (adj.) 可預測的、意料之中的。
   * prediction (n.) 預言、預測。

------------------------------
## 👑 2. contradict (v.) 反駁、與……矛盾

* 字首 contra-：反對、相對 (Against / Opposite)。
* 字根 -dict：說 (Say)。
* 字源密碼：「說反對的話、對著幹」。
* 邏輯聯想：別人說了一句話，你立刻唱反調、說相反的話去頂撞，這就是反駁；兩件事互相牴觸時，就是矛盾。
* 同家族延伸：
* contradictory (adj.) 矛盾的、對立的。

------------------------------
## 📜 3. dictate (v.) 聽寫、口述；命令、獨裁

* 字根 -dict：說 (Say)。
* 字尾 -ate：動詞字尾，表示做某個動作。
* 字源密碼：「一直在說、反覆地說」。
* 邏輯聯想：我一邊說你一邊寫，就是聽寫；高高在上的人說了算、強迫別人服從，就是命令或獨裁。
* 同家族延伸：
* dictator (n.) 獨裁者（全天下只有他能說話、發號施令的人）。
   * dictionary (n.) 字典（把所有「說出來的詞彙」收集在一起的地方）。

------------------------------
## 🏛️ 4. verdict (n.) 判決、裁決

* 字首 ver-：真實的 (True，源自拉丁文 verus)。
* 字根 -dict：說 (Say)。
* 字源密碼：「說出真實的話」。
* 邏輯聯想：法官或陪審團在法庭上，經過調查後說出最終的真相與結論，這就是判決。

------------------------------
## 💡 驚喜加碼：這個生活高頻字也是它！

* addict (v./n.) 使沉迷；上癮者
* 拆解：ad- (朝向) + dict (說) → 古羅馬時期，法庭判決把奴隸「當眾宣判交給」某主人。引申為你的心智被某個東西宣告、抓走了，完全身不由己 → 上癮。
